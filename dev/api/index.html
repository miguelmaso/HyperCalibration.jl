ExperimentsData

Data structures for defining an experimental test: the measurement, the loading/environmental protocol, the ambient conditions, and the specimen geometry. These four pieces are combined into an ExperimentData instance, which is the unit consumed by ConstitutiveModelling and ObjectiveFunctions.

HyperCalibration.ExperimentData — Type
ExperimentData{M, P, C, G}

An experiment holds a measurement M that is a function of a Protocol P -defining the independent variables- and conditionc C -environtmental static variables- and geometry G -for the reference specimen configuration-. Additionally, a weight can be assigned to the experiment to define its importance in the calibration process.

ConstitutiveModelling

Functions that evaluate a HyperFEM constitutive model against the protocol, condition, and geometry of an ExperimentData, predicting the dependent variable (stress, specific heat capacity, permittivity...) that ObjectiveFunctions compares against the measured data.

HyperCalibration.evaluate_cv — Method

Evaluate the specific heat capacity predicted by a constitutive model under the given thermal protocol, conditions and geometry of an experimental test.

source
HyperCalibration.evaluate_epsilon — Method

Evaluate the dielectric permittivity predicted by a constitutive model under the given thermal protocol, conditions and geometry of an experimental test.

source
HyperCalibration.evaluate_stress — Method

Evaluate the stress predicted by a constitutive model under the given mechanical protocol, conditions, and geometry of an experimental test.

source

Reduced kinematics

Internal kinematic maps (deformation gradient construction, thermal Jacobian solve) used by ConstitutiveModelling to reduce a 3D constitutive model to the 1D/2D loading states of a given Kinematics (uniaxial, biaxial).

HyperCalibration.F_volumetric — Method

Return the isotropic (volumetric) deformation gradient corresponding to a given Jacobian J, i.e. F = J^(1/3) * I.

source
HyperCalibration.J_thermal — Method

Return the thermally-induced volumetric Jacobian J for a given physical model and temperature. Defaults to 1.0 (no thermal expansion) for models that do not couple volume to temperature.

source
HyperCalibration.J_thermal — Method

Return the thermally-induced volumetric Jacobian for a given physical model and AbstractCondition, by extracting the temperature from the condition. Defaults to 1.0 for models that do not couple volume to temperature.

source
HyperCalibration.J_thermal — Method

Return the thermally-induced volumetric Jacobian J for a ThermalVolumetric model at a given temperature θ, solved by a Newton iteration on the zero-pressure condition pressure(J) = 0.

source
HyperCalibration.J_thermal — Method

Return the thermally-induced volumetric Jacobian for a ThermoMechano model under the given AbstractCondition.

source
HyperCalibration.calculate_F — Method

Return the purely volumetric deformation gradient for a given physical model at temperature θ, with the Jacobian solved by J_thermal.

source
HyperCalibration.calculate_F — Method

Return the deformation gradient for an independently biaxial test with stretches λ1 and λ2 under the given AbstractCondition, including any thermally-induced volumetric contribution.

source
HyperCalibration.calculate_F — Method

Return the deformation gradient for a Biaxial kinematics test at stretch λ under the given AbstractCondition, including any thermally-induced volumetric contribution.

source
HyperCalibration.calculate_F — Method

Return the deformation gradient for a Uniaxial kinematics test at stretch λ under the given AbstractCondition, including any thermally-induced volumetric contribution.

source

ObjectiveFunctions

Predefined loss functions that evaluate the mismatch between observed data and the data predicted by ConstitutiveModelling, for a given constitutive model, a set of parameters, and one or more ExperimentData. These are the functions you minimize (e.g. with Optim.jl) during calibration.

HyperCalibration.experiment_prediction — Method

Return the y_true and y_pred sampling points for a given constitutive model and experimental test.

source
HyperCalibration.loss — Method

Compute the loss function for a given constitutive model and one/multiple experimental tests as the weighted normalized squared difference between the predicted and the measured values.
